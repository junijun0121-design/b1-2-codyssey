# AI 도구 학습 #
## AI로 글·그림·영상 만들어 나만의 광고 만들기 ##
### 결과물 목차 ###
#### 1. 스토리보드(기획) 문서 ####
- 포함 항목: 브랜드 아이덴티티, 캠페인 목표/핵심 메시지, 씬 구성(씬별 필수 필드), 사용 도구 목록
- 씬별 필수 필드(모든 씬 공통):
<br>씬 번호 / 씬 길이(초)</br>
목표 메시지(한 문장)
<br>화면 구성(구도/피사체/배경/텍스트 유무)</br>
내레이션(또는 화면 카피)
<br>사용 도구(이미지/비디오/오디오)와 사용 목적</br>
입력 프롬프트(원문) + 출력 결과 요약(한 줄)
<br>생성 결과 파일명 또는 링크 표기(파일명 규칙 자유, 단 일관성 유지)</br>
- 추가 권장: 최소 1개 씬에 대해 “프롬프트 수정 전/후”와 수정 이유를 기록
#### 2. 광고 영상 파일 ####
- 길이: 10초 이내
- 영상/오디오 소스: 생성형 AI 결과물을 주된 소스로 사용(세부 제한은 ‘제약 사항’ 참고)
- 최대 인코딩 스펙(필수 아님)
해상도: 1080p
<br>프레임레이트: 24~30fps</br>
비디오 코덱: H.264
<br>오디오 코덱: AAC</br>
- 저사양/제약 허용: 해상도 720p 이상(최대 스펙 달성이 어려운 경우)
