# AI 도구 학습 #
## AI로 글·그림·영상 만들어 나만의 광고 만들기 ##

### 1. 기획 정의 ###
- 브랜드명: 기다림의 끝
- 타겟: 반려동물 입양과 유기견 문제에 관심 있는 시민
- 톤앤매너: 감성적, 따뜻함, 진정성, 시네마틱
- 차별점(USP): 유기견의 슬픔보다 “가족이 되는 순간”에 집중
- 광고 목적: 유기견을 버리지 말고 입양해 가족이 되는 따뜻함과 메시지를 전달
- 핵심 메시지: “버려진 것은 강아지가 아니라, 함께할 가족을 기다리는 생명입니다.”

<br></br>

### 2. 스토리보드 ###
- Scene 1: 유기견도 감정을 가진 생명체이며, 보호소에서 외롭게 가족을 기다리고 있다는 사실을 전달한다.
- Scene 2: 입양은 특별한 사람만 하는 일이 아니라 우리 주변의 평범한 사람도 실천할 수 있는 선택임을 보여준다.
- Scene 3: 유기견에게 필요한 것은 동정심이 아니라 평생 함께할 가족이라는 점을 강조한다.
- Scene 4: 입양은 단순히 강아지를 데려오는 행위가 아니라 한 생명의 삶을 바꾸는 결정임을 전달한다.
- Scene 5: 입양 이후의 행복한 일상을 보여주어 유기견 입양에 대한 긍정적인 이미지를 형성한다.
- Scene 6: 시청자가 광고를 보고 끝나는 것이 아니라 실제 입양에 대해 한 번 더 생각하고 행동하도록 유도한다.
<br></br>
- Scene 3 프롬프트 수정
  - 수정 전 의도: ”남자 대학생이 손을 내밀고 말티푸가 조심스럽게 다가오는 모습“
  - 문제: 남자 대학생 얼굴이 나오지 않아 교감하는 모습 부족
  - 수정 후 변경: ”앞선 인물과 동일한 모습으로 남자 대학생 얼굴이 사진에 나오기“로 프롬프트 조정
  - 결과 변화: 교감하는 장면 생성, 등장인물 감정 디테일 효과 생성

### 3. 멀티모달 생성 도구 ###
- 이미지 생성 AI
  - Dropshot(키비주얼): 유기견의 쓸쓸한 모습과 같은 디테일한 이미지를 제작해 유기견 입양이라는 주제를 직관적으로 전달할 것 같아 사용
  - Google Flow(장면 이미지): 동일한 남성 및 말티푸를 생성하는 디테일이 살아있어 장면별 흐름을 자연스럽게 연결할 수 있어 사용
- 비디오 생성 AI
  - Google Veo 3.1 Lite(16:9 영상): 프롬프트에 장면별 여러 디테일을 요구했을 때 빠뜨리지 않고, 자연스러운 움직임을 나타내서 사용
- 오디오 생성 AI
  - Suno(배경음악): 유기견의 외로움과 공익광고의 특성에 알맞은 배경음악을 생성할 수 있어서 사용
  - Supertone Play(장면별 내레이션): 공익광고 메시지를 전달하기에 알맞는 진정성 있는 톤을 생성할 수 있어 사용
- 최종 영상 생성 AI
  - Canva: 이미지, 비디오, 오디오 모든 요소를 통합해 하나의 완성도 높은 공익광고를 제작할 수 있어 사용
